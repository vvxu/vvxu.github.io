连接github

# 连接github

经常重装系统，记录一下如何快速连接github。

# 客户端生成ssh key

ssh-keygen -t rsa -C "youremail@example.com"
1

提示

  • 会提示保存路径以及要求输入密码
  • 完成以后找到路径，打开 id_rsa.pub

# github的设置

进入github个人设置，找到ssh，新建一个ssh即可。

# 验证是否正确

ssh -T git@github.com
